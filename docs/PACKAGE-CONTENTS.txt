COMMON QUESTIONS
================================================================================

Q: Do I need coding experience?
A: Basic knowledge of web development is recommended. The documentation
   provides step-by-step guidance for setup and customization.

Q: Are the third-party services free?
A: Yes! Clerk, Neon, and Stripe all offer free tiers suitable for
   development and small-scale production use.

Q: Can I customize the design?
A: Absolutely! The entire UI is built with Tailwind CSS and is fully
   customizable. See the customization guide in the documentation.

Q: Can I add my own languages?
A: Yes! Use the admin panel to create courses, units, lessons, and
   challenges for any language you want to teach.

Q: Is this ready for production?
A: Yes! This is a production-ready application. Just configure your
   production environment variables and deploy.

Q: Can I remove the sample data?
A: Yes! Run `npm run db:reset` to clear all data and start fresh,
   or use the admin panel to modify/delete content.

Q: Do I need to use all features?
A: No! You can disable features you don't need. For example, you can
   skip Stripe integration if you don't want paid subscriptions.

Q: Can I use this for other types of learning?
A: Yes! While designed for languages, you can adapt it for teaching
   anything that works with a question/answer format.
